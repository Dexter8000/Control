Agregar Nuevo Usuario

La creación y edición de usuarios se realiza mediante SQLite Studio u otras herramientas de administración directa de la base de datos.

Este panel solo permite visualizar la información de los usuarios existentes.

Para agregar un nuevo usuario, utilice las herramientas de administración de base de datos.

Estructura de la tabla de usuarios
Campo Tipo Descripción
id INTEGER Identificador único (PRIMARY KEY)
usuario TEXT Nombre de usuario para acceso
password TEXT Contraseña (almacenada con hash)
rol TEXT Rol del usuario (administrador, usuario, etc.)
nombre TEXT Nombre real del usuario
apellido TEXT Apellido del usuario
email TEXT Correo electrónico
telefono TEXT Número de teléfono
activo INTEGER Estado del usuario (1=activo, 0=inactivo)
fecha_creacion TEXT Fecha de creación del usuario
ultimo_acceso TEXT Fecha del último acceso
intentos_fallidos INTEGER Número de intentos fallidos de inicio de sesión
bloqueado_hasta TEXT Fecha hasta la que el usuario está bloqueado
